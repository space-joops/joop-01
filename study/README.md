# 🎓 줍스로 배우는 웹 프론트엔드 — 5일 마스터 플랜

> 파이썬 백엔드/데이터 엔지니어를 위한 실전 커리큘럼.
> 교재는 다름 아닌 **이 저장소** — 여기서 배우는 모든 개념은 우리가 직접 짠
> 코드에 살아 있고, 파일 경로가 함께 적혀 있어 바로 열어 확인할 수 있습니다.

## 왜 이 프로젝트가 최고의 교재인가

튜토리얼 앱(Todo 리스트)에는 없는 것들이 여기 다 있습니다:

| 축 | 이 프로젝트의 실전 사례 |
| --- | --- |
| 렌더링 아키텍처 | Next.js App Router, Server/Client Component, hydration |
| 상태 관리 | Zustand 스토어 + localStorage persist + 서버 동기화 2단 구조 |
| 애니메이션 | Framer Motion 선언형 + rAF 명령형 게임 루프 공존 |
| 실시간 그래픽 | 60fps 미니게임, 관성 물리, 베지에 곡선, Lottie |
| 플랫폼 | PWA(서비스 워커·웹 푸시·설치), 모바일 뷰포트(dvh·safe-area) |
| 백엔드 연동 | Supabase 익명 인증·RLS·security definer RPC, Server Actions |
| 품질 | 번들 -48% 다이어트 실화, Playwright 자동 검증, OG/공유 |

## 학습 방법 (중요 — 이대로 하세요)

1. **아침(1h): 읽기.** 그날 문서를 처음부터 끝까지. 파이썬 비유가 나오면
   "아 그거"라고 소리 내어 번역해 보세요.
2. **낮(2~3h): 코드 열기.** 문서에 적힌 파일 경로를 전부 열어 실제 코드와
   개념을 대조합니다. `npm run dev` 켜 두고 값을 바꿔 보며 화면으로 확인.
3. **저녁(1~2h): 손 실습.** 각 문서 끝의 실습 과제를 **직접 타이핑**으로.
   복붙은 근육이 안 생깁니다.
4. **자기 전(10m): 체크리스트.** 각 문서의 셀프 체크에 답할 수 없으면
   그 절만 다시.
