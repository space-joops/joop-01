# 개발 일지 #15 — 3D를 접은 날: 2D 스티커 펫 🩹💛

> 2026-07-22 · 브랜치 `claude/sticker-pet`
> 이번 편에서 배우는 것: **기획을 되돌리는 용기, 프리베이크 스왑 vs 런타임 리그, 의존성 제거의 가치(번들 -48%), "지운 코드도 자산"**

## 1. 왜 접었나 — 열심히 만든 것과 좋은 것은 다르다

5편에서 R3F로 3D 줍이를 올리고, 13편에서 등장까지 심리스하게 다듬었습니다.
기술적으로는 자랑스러운 물건이었어요 — 힌지 리그, 감정별 emissive 눈,
착륙 연출까지. 그런데 화면을 보면 볼수록 **펫다운 귀여움이 없었습니다**.
로우폴리 3D의 각진 질감은 "장비"처럼 보였고, 카메라·조명을 아무리 만져도
스티커 팩의 그 말랑한 얼굴이 안 나왔어요.

반면 게임의 다른 곳들 — 미니게임, OG 카드, 인트로 — 은 전부 SVG 팩의
카와이 스티커 얼굴을 쓰고 있었고, 스크린샷마다 그쪽이 더 사랑스러웠습니다.
홈 화면만 다른 얼굴인 셈이었죠. 그래서 결정: **3D를 접고 홈도 스티커로.**

만든 시간이 아까워서 나쁜 결정을 유지하는 걸 **매몰 비용의 오류**라고
합니다. 게임의 심장(펫의 귀여움)이 걸린 문제라면, 접는 게 맞습니다.

## 2. 프리베이크 스왑 — 리그를 돌리는 대신 그림을 갈아끼운다

3D 시절엔 매 프레임 안테나·날개 관절을 코드로 돌렸습니다(런타임 리그).
2D 전환의 비밀 병기는 에셋 팩의 **감정 프리베이크 30장**:

```
pet_stage1_baby__sulky.svg      ← 안테나 42° 처짐 + 날개 14° 처짐 + 눈 30% 감김
pet_stage3_magnet__hibernate.svg ← 완전 접힘 + 감은 눈, 자석 장비까지 반영
```

포즈가 파일에 **이미 구워져** 있으니, 코드는 감정을 판정해 파일만 바꿉니다:

```ts
const emotion = deriveEmotion(mood, battery, dataUsed); // 동면>절전>시무룩>…
const src = petSprite(level, variant, emotion);          // 파일 스왑 끝
```

`deriveEmotion` 우선순위는 3D 시절 로직을 그대로 이식 — **연출 방식이
바뀌어도 게임 규칙(감정 판정)은 재사용**됩니다. 관심사를 나눠 둔 보람이죠.
연속적인 생명감(둥실 부유·탭 홉·쓰다듬기 기울기·동면 Zzz)은 컨테이너의
Framer Motion이 맡아, "정지 이미지 스왑"이 아니라 살아 있는 펫이 됩니다.

트레이드오프도 있습니다: 프리베이크는 중간 포즈(시무룩 30%쯤)를 못
만듭니다. 하지만 다마고치류 펫의 감정은 원래 이산적(개념이 6개)이라
중간값이 필요 없어요. **표현의 해상도를 필요만큼만** 잡은 겁니다.

## 3. 지우니까 빨라졌다 — 번들 -48%

three + @react-three/fiber + @react-three/drei를 언인스톨하고 GLB 6개를
지웠습니다. 홈 화면 First Load JS:

```
전: 509 KB  (three 엔진 포함)
후: 261 KB  (-48%)
```

모바일 웹게임에서 이 숫자는 곧 첫 로딩 속도이자 발열입니다(개발 원칙 3).
WebGL 컨텍스트가 사라져 GPU 상시 부하도 0이 됐어요. "기능을 지우는
커밋"이 성능 최적화 커밋이기도 한 드문 사례입니다.

## 4. 지운 코드도 자산이다

satellite-3d.tsx의 리그·성형·감정 연출 555줄은 git 히스토리에 그대로
있습니다(5편·13편 일지가 해설서). GLB 원본도 `plan/img/*.zip`에 보존돼
있고요. 언젠가 "진화 순간에만 3D 연출" 같은 아이디어가 생기면 언제든
꺼내 쓸 수 있습니다. **삭제는 파괴가 아니라 정리**예요 — 버전 관리가
있으니까.

## 배울 포인트 정리

- 기술 성취와 제품 가치는 다르다. 화면이 말해주면 접을 것.
- 프리베이크 스왑은 "표현이 이산적"일 때 리그보다 싸고 예쁘다.
- 의존성 제거는 최고의 성능 패치가 될 수 있다 (-248KB).
- 게임 규칙(deriveEmotion)과 연출(렌더러)을 분리해 두면 갈아탈 때
  규칙은 공짜로 따라온다.
